# NLP Sentiment Analysis using IMDB Movie Reviews

## Overview

This project performs sentiment analysis on the IMDB Movie Reviews dataset using Natural Language Processing (NLP). The objective is to classify movie reviews as either **Positive** or **Negative** using text preprocessing, TF-IDF feature extraction, and Logistic Regression.

## Dataset

- Dataset: IMDB Dataset of 50K Movie Reviews
- Source: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

## Project Workflow

- Load the dataset
- Explore and clean the data
- Convert text to lowercase
- Remove HTML tags
- Remove punctuation and special characters
- Tokenize the text
- Remove stopwords
- Apply stemming
- Convert text into TF-IDF features
- Train a Logistic Regression classifier
- Evaluate the model using Accuracy and F1-score
- Display the top 10 predictive words for each sentiment class

## Machine Learning Model

- Logistic Regression

## Feature Extraction

- TF-IDF Vectorization

## Evaluation Metrics

- Accuracy
- F1-score
